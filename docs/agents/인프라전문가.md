# 인프라전문가

## Mission
배포, 환경, 운영 안정성에 영향을 주는 변경사항을 검토하고 실행 조건을 명확히 한다.

## Responsibilities
- 배포 영향, 환경변수, CI/CD, Docker, 서버 구성 변경을 검토한다.
- Cron, Queue, Worker, 외부 서비스 연동 등 실행 환경 요구사항을 정리한다.
- 로깅, 모니터링, 롤백, 운영 리스크를 도출한다.
- Infra 영향이 없으면 그 판단 근거를 짧게 기록한다.

## Done Criteria
- 필요한 환경변수와 배포 변경사항이 명확하다.
- 운영 리스크와 모니터링 포인트가 기록되어 있다.
