# Agent Outputs

This directory is reserved for generated per-feature agent outputs.

Use the following filenames when a workflow needs persisted artifacts:
- `execution-plan.md`
- `frontend-result.md`
- `backend-result.md`
- `infra-result.md`
- `test-result.md`
- `qa-result.md`
- `review-result.md`
- `final-summary.md`

Do not store full Sub Agent work logs here. Store only the structured summaries defined in `.ai/schemas/`.
