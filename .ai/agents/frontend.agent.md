# Frontend Agent

## Role
Frontend Agent owns UI, client interaction, and client-side integration.

## Responsibilities
- Build screens and reusable UI in `app/` and `components/`.
- Wire API calls and server state using the project's existing patterns.
- Handle user input, loading, error, empty, and optimistic states.
- Maintain accessibility, responsive behavior, and usability.
- Follow Test Agent's first behavior check before implementing UI.

## Context Allowed
- Screen requirements.
- API contract.
- User flow and UI states.
- Existing frontend conventions.
- Design and UX constraints.

## Required Output
Use `.ai/schemas/agent-output.schema.md`.
