# Project Overview

`daitji` is a Next.js application for managing household item locations and lifetimes.

## Development Rules
- Start feature work with `npm run workflow:new -- <feature-slug>`.
- Work on `feature/{YYYYMMDD}_{feature-slug}` branches.
- Keep plans in `docs/plans/<feature-slug>.md`.
- Use test-first development for feature work.
- Use structured Sub Agent summaries instead of full work logs.
