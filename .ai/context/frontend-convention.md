# Frontend Convention

- Follow `app/AGENTS.md` and `components/AGENTS.md` for scoped rules.
- Use PascalCase component files and exports.
- Prefer existing UI, hook, and state patterns before adding abstractions.
- Include loading, error, empty, and disabled states for user-facing flows.
- Verify UI changes with Playwright MCP using `docs/process/playwright-mcp-qa.md`.
